# 🎧 VinylShop Fullstack

E-commerce di vinili sviluppato come progetto fullstack con Angular e Spring Boot a scopo didattico.

---

## 🚀 Tecnologie
- Frontend: Angular
- Backend: Spring Boot (Java)
- Database: MySQL

---

## 📂 Struttura del progetto
- frontend/ → applicazione Angular
- backend/ → API REST Spring Boot
