# VinylShop Fullstack

E-commerce di vinili sviluppato con:

## 🔧 Tecnologie
- Frontend: Angular
- Backend: Spring Boot (Java)
- Database: MySQL

## 📂 Struttura
- frontend/ → applicazione Angular
- backend/ → API REST Spring Boot

## 🚀 Funzionalità
- Registrazione / Login
- Visualizzazione vinili
- Carrello
- Ordini
- Area admin
- Area utente

### Frontend
```bash
cd frontend/vinylshop-fe
